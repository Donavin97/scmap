I built scmap — a high-resolution seismic event map generator for SeisComP.

It reads SCML (SeisComP Markup Language) files or queries a live SeisComP database and renders publication-quality maps with Cartopy and matplotlib. Four map modes: event markers with depth colouring, Gutenberg-Richter b-value heatmaps, magnitude-of-completeness (MAXC), and annual seismicity rate maps.

Design follows established conventions (GMT, USGS, GFZ) — perceptually uniform colour scales, colourblind-safe markers, reference magnitude circles, density-aware city label placement, and a clean low-data-ink basemap.

Built as a proper seiscomp.client.Application so it inherits all standard SeisComP options (-d, -H, --debug, --log-file).
